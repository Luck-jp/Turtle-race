🐢 Turtle Race Game (Python)

A fun graphical turtle racing game where the user bets on which turtle will win the race.

🚀 Features
Multiple turtles with different colors
User betting system
Random movement for each turtle
Graphical interface using Turtle

🛠️ Tech Used
Python 3
Turtle (built-in)

▶️ How to Run

Clone the repository:
git clone https://github.com/Luck-jp/turtle-race-python.git
Navigate to the folder:
cd turtle-race-python
Run the program:
python main.py

🎮 How It Works

User selects a turtle color
Race begins
Turtles move randomly
First turtle to reach the finish line wins

📌 Project Purpose

This project was built to understand:

Turtle graphics
Randomization
Event-driven interaction
Basic game logic
